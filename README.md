# PyMongo Client

Ejemplo de conexión de un cliente MongoDb en Ubuntu 18.04

## Instalaciones

```bash
sudo apt-get install mongodb -y
```

```bash
sudo pip install pymongo
```

## Ejecución

```bash
sudo python src/index.py
```
