# PyMongo Client

Ejemplo de conexión de un cliente MongoDb en Ubuntu 18.04 y Python 3.8

## Instalación

Paquete para Ubuntu

```bash
sudo apt-get install mongodb -y
```

Módulo para Python 3

```bash
sudo pip install pymongo
```

## Ejecución

```bash
sudo python src/index.py
```
